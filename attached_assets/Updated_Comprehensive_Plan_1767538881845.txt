Data Structure (Airtable Tables)
1. Evidence Categories

Name (e.g., "Identification", "Income", "Assets")
Display Order
Color (optional, for visual grouping)
2. Evidence Templates (Master list)

Name (rich text)
Description (rich text)
Category (linked to Evidence Categories)
Display Order
Opportunity Types (multiple select: Home Loans, Commercial, Insurance, etc.)
Is Lender Specific (checkbox) - if true, only appears when a lender rule exists
3. Lender Evidence Rules (Junction table for lender variations)

Evidence Template (linked)
Lender (text or linked if you have a Lenders table)
Rule Type (single select: "Required", "Not Required", "Custom Description")
Custom Description (rich text) - overrides the template description for this lender
Custom Name (rich text, optional) - if the item name differs for this lender
4. Evidence Items (Per-opportunity instances)

Opportunity (linked)
Name (rich text)
Description (rich text)
Notes (rich text, internal only)
Category (text, copied from template)
Display Order
Status (single select: Outstanding, Received, N/A)
Date Received (date/time)
Requested On (date/time) - auto-set when included in Initial/Subsequent email
Requested By (linked to Users)
Is Custom (checkbox) - true if manually added, not from template
Source Template (linked to Evidence Templates, for reference)
Created On, Created By, Modified On, Modified By (audit)
5. Email Templates (Enhancement to existing)

Add "Evidence Initial Request" template
Add "Evidence Subsequent Request" template
Add "Appointment Confirmation" template (with evidence list option)
